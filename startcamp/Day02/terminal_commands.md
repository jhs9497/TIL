# Bash Shell 기초 명령어



## 1.  pwd

```bash
$ pwd
```



## 2. ls

```bash
// 1. 기본 명령어
$ ls

// 2. 숨겨진 파일까지 보기
$ ls -a
```



## 3. clear

```bash
$ clear
```



## 4. cd

>cd는 경로 이동 시 사용하는 명령어입니다.

```bash
// 1. 기본 명령어
$ cd 이동하고자_하는_폴더명_또는_경로

// 2. 상위 폴더로 이동하기
$ cd ..

// 3. 이전 경로로 다시 돌아가기
$ cd -
```
